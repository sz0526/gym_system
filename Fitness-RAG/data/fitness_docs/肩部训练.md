# 肩部训练（三角肌）专项指南

## 核心目标肌群
三角肌分为前束、中束和后束。完美的肩部需要三束均衡发展，形成球形肩部。

## 经典训练动作

### 动作一：哑铃坐姿推举 (Dumbbell Shoulder Press)
* **主要目标肌群**：三角肌前束、三角肌中束。
* **动作要领**：
  1. 坐在上背靠紧的训练凳上，双手各持一只哑铃，将其提起至肩部两侧，手心朝前。
  2. 呼气，三角肌发力将哑铃向上推起，直至手臂接近伸直（不要锁死肘关节）。
  3. 在最高点感受肩部受力，随后吸气，缓慢将哑铃下放至耳垂高度。
* **常见错误**：腰部过度向前拱起导致胸肌上部借力、哑铃在中途相撞。
* **推荐组数**：4组 x 8-12次。

### 动作二：哑铃侧平举 (Dumbbell Lateral Raise)
* **主要目标肌群**：三角肌中束（增加肩部宽度、打造太平洋宽肩）。
* **动作要领**：
  1. 保持站姿，挺胸收腹，双手各持哑铃垂于身体两侧，手心相对。
  2. 肘关节微屈，呼气，以肩部为轴，将哑铃向身体两侧上方外展抬起，直至手臂与地面平行。
  3. 此时小拇指位置略高于大拇指（呈倒水状）。吸气，缓慢控制哑铃下放还原。
* **常见错误**：利用身体惯性甩动重量、耸肩导致斜方肌过度代偿。
* **推荐组数**：4组 x 12-15次。

### 动作三：俯身哑铃飞鸟 (Bent-Over Dumbbell Rear Delt Fly)
* **主要目标肌群**：三角肌后束（消除圆肩、让肩膀更立体）。
* **动作要领**：
  1. 双脚并拢或与肩同宽，屈髋俯身至上半身几乎与地面平行，保持背部挺直。
  2. 双双手各持哑铃自然下垂，掌心相对，肘关节微屈。
  3. 呼气，三角肌后束发力，将哑铃向身体两侧后上方抬起，整个过程保持肘关节角度不变。
  4. 抬至上臂与后背平齐，吸气控制下放还原。
* **常见错误**：大臂往后夹变成了练背（过度夹肩胛骨）、抬头导致颈椎压力过大。
* **推荐组数**：4组 x 12-15次。

### 动作四：杠铃立姿划船 (Barbell Upright Row)
* **主要目标肌群**：三角肌中束、斜方肌。
* **动作要领**：
  1. 自然站立，双手正握杠铃，握距略宽于肩（较宽的握距能减少斜方肌发力，更多刺激中束）。
  2. 挺胸收腹，呼气，三角肌发力，提起双肘带动杠铃沿身体前侧垂直向上拉起。
  3. 杠铃拉至接近胸肌中部（下巴下方）即可，此时肘关节应高于前臂。
  4. 吸气，控制重量缓慢下放。
* **常见错误**：握距过窄导致手腕剧痛、拉得太高（超过锁骨）容易引发肩峰撞击。
* **推荐组数**：4组 x 10-12次。

### 动作五：蝴蝶机反向飞鸟 (Reverse Machine Flye)
* **主要目标肌群**：三角肌后束（极佳的后束孤立动作）。
* **动作要领**：
  1. 面对蝴蝶机坐好，调整座椅高度使把手与肩同高。胸口紧贴靠垫。
  2. 双手正握或对握把手，肘关节微屈并保持固定。
  3. 呼气，三角肌后束发力，将双臂向后水平外展画弧线，直至双臂与身体呈一条直线。
  4. 顶峰收缩1秒，吸气，控制重量缓慢还原，注意不要让配重片相撞。
* **常见错误**：身体借力前后晃动、手肘弯曲角度改变导致三头肌发力。
* **推荐组数**：4组 x 15次。
