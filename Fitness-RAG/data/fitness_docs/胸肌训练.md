# 胸部肌群针对性训练计划

## 核心目标肌群
胸大肌（上胸、中部、下胸）以及前锯肌。

## 经典训练动作

### 动作一：杠铃平卧推举 (Bench Press)
* **主要目标肌群**：胸大肌中部（整体厚度）、三角肌前束、肱三头肌。
* **动作要领**：
  1. 仰卧在平板凳上，头部、上背部和臀部紧贴凳面，双脚平放于地面。
  2. 双手以略宽于肩的握距握住杠铃，将其移出卧推架。
  3. 吸气，控制杠铃缓慢下放，直至杠铃杆轻触胸肌下沿（乳头附近）。
  4. 呼气，胸肌发力将杠铃垂直向上推起至初始位置，注意收紧核心。
* **常见错误**：双脚离地晃动、手腕过度后翻导致手腕受伤、肘关节向外过度外展（伤肩）。
* **推荐组数**：5组 x 6-10次。

### 动作二：哑铃上斜飞鸟 (Incline Dumbbell Fly)
* **主要目标肌群**：胸大肌上部（修饰锁骨下方线条）、胸肌中缝。
* **动作要领**：
  1. 躺在倾斜角约为30-45度的上斜凳上，双手持哑铃伸直在胸部上方，掌心相对。
  2. 保持肘关节微屈，吸气，两臂张开向身体两侧画弧线缓慢下放，直至胸肌有充分的拉伸感。
  3. 呼气，胸肌发力带动双臂沿着原弧线轨迹向内夹回，想象去拥抱一棵大树，在顶峰时胸肌强烈收缩。
* **常见错误**：下放得太深伤害肩关节、手臂完全伸直变成推举。
* **推荐组数**：4组 x 12次。

### 动作三：双杠臂屈伸 (Chest Dips)
* **主要目标肌群**：胸大肌下沿、肱三头肌、三角肌前束。
* **动作要领**：
  1. 双手撑在双杠上，身体悬空，上身微微前倾（约30度），双腿向后弯曲交叠。
  2. 吸气，缓慢弯曲手臂下放身体，直到上臂约与地面平行，感受胸肌下沿的拉伸。
  3. 呼气，胸肌下沿与肱三头肌同时发力，将身体推回初始位置。
* **常见错误**：身体过于垂直（导致力量转移到三头肌）、下放过深导致肩关节压力过大。
* **推荐组数**：4组 x 8-12次。

### 动作四：上斜哑铃推举 (Incline Dumbbell Press)
* **主要目标肌群**：胸大肌上部（饱满上胸）。
* **动作要领**：
  1. 坐在30-45度角的上斜凳上，双手各持哑铃置于大腿上。
  2. 顺势将哑铃推举至肩部外侧，掌心朝前。
  3. 呼气，胸肌上部发力将哑铃垂直向上推起，在最高点不要锁死肘关节。
  4. 吸气，控制重量缓慢下放至胸口两侧，感受上胸拉伸。
* **常见错误**：凳子角度过高（超过45度会变成练肩）、哑铃下放时撞击胸部。
* **推荐组数**：4组 x 8-12次。

### 动作五：绳索低位到高位夹胸 (Low-to-High Cable Flye)
* **主要目标肌群**：胸大肌上部、胸肌中缝内侧。
* **动作要领**：
  1. 将龙门架的滑轮固定在接近地面的低位，双手各握住一个手柄。
  2. 往前走一步，呈弓步站立，躯干微微前倾，双臂在身体两侧微屈。
  3. 呼气，胸肌发力带动双臂由下往上、往内弧线聚拢，直至双手手背在眼睛前方高度靠拢。
  4. 吸气，控制重量缓慢沿原轨迹退回。
* **常见错误**：借助身体前晃的惯性拉起重量、耸肩。
* **推荐组数**：4组 x 12-15次。
