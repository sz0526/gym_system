# 背部训练肌群与动作指南

## 核心目标肌群
背部训练主要针对背阔肌、斜方肌、大圆肌、小圆肌以及竖脊肌。

## 经典训练动作

### 动作一：高位下拉 (Lat Pulldown)
* **主要目标肌群**：背阔肌外侧（打造倒三角视觉）、大圆肌。
* **动作要领**：
  1. 坐在下拉机凳上，调整挡腿板固定双腿，双手采取比肩稍宽的握距正握把手。
  2. 挺胸收腹，身体微微后仰（约10-15度）。
  3. 呼气，用背阔肌发力带动肘关节向下、向后拉动把手，将把手拉至接近上胸部位置。
  4. 在顶峰收缩时停顿1-2秒，感受背部肌肉的挤压。
  5. 吸气，用背阔肌控制着重量缓慢还原。
* **常见错误**：耸肩、身体前后过度晃动、过度借用手臂二头肌发力。
* **推荐组数**：4组 x 10-12次。

### 动作二：俯身杠铃划船 (Barbell Row)
* **主要目标肌群**：背阔肌、中斜方肌、菱形肌（增加背部厚度）。
* **动作要领**：
  1. 双脚平稳站立与肩同宽，双手正握杠铃，握距略宽于肩。
  2. 屈髋屈膝，上身前倾与地面呈约45度角，保持背部挺直，挺胸核心收紧。
  3. 呼气，背部发力，将杠铃沿着大腿顺势拉向肚脐方向。
  4. 感受背部肌肉完全收缩，随后吸气控制重量缓慢下放至初始位置。
* **常见错误**：弓背（极易导致腰部受伤）、站立姿势太直导致变成耸肩。
* **推荐组数**：4组 x 8-10次。

### 动作三：引体向上 (Pull-Up)
* **主要目标肌群**：背阔肌（王牌宽背动作）、大圆肌、核心肌群。
* **动作要领**：
  1. 双手以略宽于肩的握距正握单杠，身体自然悬垂，核心收紧。
  2. 呼气，预先启动下拉肩胛骨（沉肩），随后背阔肌发力将身体向上拉起。
  3. 尽量拉至下巴超过单杠，或者上胸贴近单杠。
  4. 吸气，控制身体缓慢下放还原，至手臂接近伸直。
* **常见错误**：完全靠手臂死拉、身体过度晃动、没有沉肩导致斜方肌代偿。
* **推荐组数**：4组 x 力竭（或 8-10次）。

### 动作四：坐姿绳索划船 (Seated Cable Row)
* **主要目标肌群**：中下斜方肌、菱形肌、背阔肌内侧（雕刻背部细节与厚度）。
* **动作要领**：
  1. 面向器械坐好，双脚踩在踏板上，微屈膝，双手握住V字把手。
  2. 挺胸塌腰，手臂伸直，身体垂直于地面或微微后仰。
  3. 呼气，肩胛骨后缩，背部发力将把手拉向腹部（肚脐上方）。
  4. 肘关节紧贴身体两侧向后拉到底，顶峰收缩1秒。
  5. 吸气，缓慢释放重量，保持躯干稳定。
* **常见错误**：骨盆后倾弓背、拉重量时身体大幅度前后晃动。
* **推荐组数**：4组 x 10-12次。

### 动作五：直臂下压 (Straight-Arm Pulldown)
* **主要目标肌群**：背阔肌下部（有效孤立背阔肌，不涉及二头肌发力）。
* **动作要领**：
  1. 面对龙门架，双手正握直杆或绳索，双脚与肩同宽，髋关节微屈，躯干微前倾。
  2. 肘关节保持微屈并锁死不变，双手持杆高举。
  3. 呼气，背阔肌发力，将直杆呈弧线向下、向后压至大腿前侧。
  4. 在最底端用力挤压背阔肌，随后吸气，控制重量缓慢还原。
* **常见错误**：肘关节弯曲变成了三头肌下压、耸肩。
* **推荐组数**：4组 x 12-15次。
